## 3. 训练数据说明

### 3.1 模型训练数据

**梗星球** 不自行训练/微调基础模型。使用的模型均为供应商已备案模型，训练数据由供应商负责合规。

### 3.2 平台自有数据

平台内部数据（梗卡内容、评分、评论、用户兴趣标签）仅用于：

| 用途 | 数据范围 | 说明 |
| --- | --- | --- |
| **RAG 知识库** | 已发布的公开梗卡 | 向量化存入 pgvector，用于造梗 Agent 检索相似梗（预留 M2 实现） |
| **LLM-as-Judge 评分** | 梗卡内容 + 评分 | v1.5 评估，不涉及 M1 |
| **模型微调** | 无 | M1 不进行任何模型微调，v2.0 评估时单独申报 |
| **推荐系统** | 用户兴趣标签 + 热度分 | 基于规则的召回（v1），不涉及用户画像模型的训练 |

### 3.3 数据合规承诺

1. 不使用用户个人数据（手机号、设备信息等）用于模型训练
2. 所有 AI 生成内容的 prompt + 输出留存 6 个月备查
3. 用户可申请删除个人数据（详见隐私政策）
4. 不使用爬取/侵权数据
5. Prompt 模板知识：5 类官方模板（抽象/阴阳/谐音/反转/表情包配文）为产品团队原创设计，无版权风险

## 4. 安全评估报告

### 4.1 安全评估结论

梗星球（MemeChatAI）对 AI 生成内容实施**"生成前 + 发布前"双重安全审核**，配合**敏感词 DFA + 阿里云内容安全 + 机审队列 + 人工抽审"四级过滤机制，确保 AI 生成内容符合法律法规和社会主义核心价值观。

**安全评估等级**：中风险（AI 生成内容面向公众传播，但全部经过审核+标识+可追溯）

### 4.2 安全风险识别与缓解

| 风险类型 | 风险等级 | 缓解措施 |
| --- | --- | --- |
| 生成涉政/色情/暴力内容 | 高 | 机审（DFA + 阿里云内容安全）+ 人审队列 + 熔断机制 |
| 提示词注入 | 中 | prompt 模板变量隔离 + 用户输入不直接操纵系统 prompt |
| 未成年人接触不适宜内容 | 高 | 青少年模式 + AI 生成标识 + 年龄字段（M2 强制实名评估） |
| 版权侵权 | 中 | 图像/视频不使用受版权保护的人物形象；文本原创 |
| 深度合成滥用 | 高 | MVP 不开放人脸/声音克隆（详见 §10） |
| 数据泄露 | 中 | 日志脱敏 + 数据 6 个月自动清理 + 不传输个人敏感信息 |
| 恶意刷 API | 中 | 单用户限频 10 次/日 + 能量机制 + 熔断 |
| 军团刷分/僵尸号 | 中 | 同军团评分降权 + 异常检测 |

### 4.3 应急预案

| 事件 | 响应 | 恢复 |
| --- | --- | --- |
| 大量违规内容生成 | 自动熔断 AI 生成（日预算/配额超限触发） | 人工审查后逐步恢复 |
| 模型服务中断 | 自动降级到 fallback 模型（DeepSeek→GLM→Mock） | 主模型恢复后自动切回 |
| 审核系统失效 | 切换到"全部人审"模式（阻塞发布） | 审核恢复后逐步放行 |
| 内容安全事件（监管通报） | 全量下架涉事内容 + 配合调查 + 暂停 AI 生成 | 合规整改通过后恢复 |

## 5. 内容审核机制

### 5.1 四级审核体系

```
用户输入
   │
   ▼
┌────────────────────────────────┐
│ 第一级: DFA 敏感词库           │ ← 本地 Trie 匹配，≈50 内置词
│ 命中 → 立即拒绝              │    热更新，拼音变体/干扰字符跳过
└────────────┬───────────────────┘
   │ 通过
   ▼
┌────────────────────────────────┐
│ 第二级: 阿里云内容安全         │ ← 文本审核 SDK（M2 接入图片）
│ 违规 → 拒绝                  │    阈值可配置
└────────────┬───────────────────┘
   │ 通过
   ▼
┌────────────────────────────────┐
│ 第三级: 机审队列 (BullMQ)      │ ← 异步队列，自动审核
│ 可疑 → 人审队列              │    published / manual_review / rejected
└────────────┬───────────────────┘
   │ 通过
   ▼
┌────────────────────────────────┐
│ 第四级: 人工审核 (后台)        │ ← 运营后台人审入口（M2）
│ 最终决定                     │    驳回 + 通知
└────────────────────────────────┘
   │
   ▼
  发布 (status = published)
```

### 5.2 审核范围

| 内容类型 | 审核方式 | 处罚措施 |
| --- | --- | --- |
| 梗卡标题/正文 | DFA + 阿里云内容安全 | 命中敏感词：自动拒绝；机审可疑：入人审 |
| 用户评论 | DFA + 阿里云内容安全 | 命中敏感词：拦截+警告 |
| 军团名称/简介 | DFA（创建时审核） | 命中敏感词：禁止创建 |
| 用户昵称/签名 | DFA（修改时审核） | 命中敏感词：拦截 |
| 头像/图片 | 阿里云内容安全（M2 接入） | 违规：隐藏+通知 |

### 5.3 人工审核

- M2 完善运营后台人审入口
- 人审队列基于 BullMQ，优先级标记，支持批量操作
- 审核结果通知用户（系统通知）
- 审核记录全量留存（operator_id + action + reason + timestamp）

### 5.4 用户举报机制

- 梗卡/评论/用户支持举报（M1 基础功能）
- 举报类型：涉政、色情、暴力、侵权、垃圾广告
- 举报入审核队列，按优先级处理

| 位置 | 标识方式 | 实现状态 |

- 字体清晰可辨，不被遮挡
- 位置显著（详情页正文上方 / 卡片角标 / 视频角落）

## 7. 用户协议与隐私政策要点

> 详见 `docs/context/privacy-policy.md`（T4.2 产出）和 `docs/context/user-agreement.md`（T4.2 产出）

### 7.1 AI 生成内容条款（必须包含）

1. **内容权利声明**：用户对 AI 生成内容拥有使用权，平台拥有平台内展示权
2. **免责声明**：AI 生成内容仅供参考，平台不保证生成内容的准确性、合法性
3. **禁止事项**：利用 AI 生成违法/侵权/违背公序良俗的内容
4. **AI 标识声明**：所有 AI 生成内容均带有"AI 生成"标识
5. **数据使用**：用户输入（关键词/prompt）仅用于当前生成请求，不用于模型训练
6. **赔偿条款**：用户违规使用 AI 功能导致的法律后果由用户承担

### 7.2 未成年人条款

1. **年龄限制**：16 岁以下用户需监护人同意
2. **青少年模式**：默认关闭，开启后限制使用时间（40min/日）、时段（22:00–06:00 禁用）、功能限制（禁发布/评分/私信）
3. **实名认证**：M2–M3 评估强制实名

## 10. 深度合成限制声明

根据《互联网信息服务深度合成管理规定》，梗星球（MemeChatAI）**MVP 阶段不开放以下功能**：

| 被禁止的功能 | 原因 |
| --- | --- |
| 人脸生成/替换 | 需额外合规审批 + 显著标识 + 被替换人授权 |
| 声音克隆/音色定制 | 需额外合规审批 + 被克隆人授权 |
| 人脸特征提取 | 生物特征属于敏感个人信息 |
| 面部/姿态驱动动画 | 需额外合规审批 |
| 实时换脸/深度伪造 | 严格禁止 |

**确认承诺**：
- MVP 不实现上述任何功能
- 视频生成仅使用「关键词 + 预设风格」的纯文生视频（抽象/风格化内容）
- TTS 仅提供 4 种预设标准音色（不涉及声音克隆）
- 未来如需开放上述功能，将单独提交深度合成备案申请

**配合监管**：如发现用户试图绕过限制使用深度合成技术，平台将立即封禁并上报。
