## 1. 法规依据

| 法规 | 关键要求 | 本产品对应措施 |
| --- | --- | --- |
| 《未成年人保护法》(2021修订) | 网络产品提供者须设置青少年模式 | 青少年模式入口在我的页，可一键开启 |
| 《未成年人网络保护条例》(2024施行) | 限制使用时长/时段、功能限制 | 40min/日、22-06 禁用、功能限制 |
| 《生成式AI服务管理办法》 | 不得向未成年人提供不符合其身心发展的服务 | 青少年模式下禁止 AI 生成/发布 |
| 《儿童个人信息网络保护规定》 | 收集儿童信息需监护人同意 | 14-18 岁需确认年龄；14 岁以下禁用 |

## 2. 年龄分层策略

| 年龄段 | 注册权限 | 服务范围 | 特殊要求 |
| --- | --- | --- | --- |
| < 14 周岁 | 禁止注册 | 不可使用本服务 | 检测到注册尝试时提示"本服务不对14周岁以下未成年人开放" |
| 14-17 周岁 | 允许注册 | 受限服务（默认开启青少年模式） | 注册时需输入出生日期；系统自动判定并开启青少年模式 |
| ≥ 18 周岁 | 允许注册 | 完整服务 | 标准注册流程 |

### 2.1 年龄识别方式

**M1 阶段（基础）**：
- 注册流程中增加出生日期输入（月/日/年选择器）
- 系统根据出生日期自动计算年龄
- 14-17 岁自动开启青少年模式

**M3 阶段（强化）**：
- 考虑接入身份证实名认证（需评估隐私合规成本）
- 行为分析检测异常低龄用户

## 3. 青少年模式详细规则

### 3.1 时间限制

| 规则 | 内容 |
| --- | --- |
| 每日使用时长 | 40 分钟 |
| 倒计时提醒 | 距离 40 分钟还剩 5 分钟时弹窗提醒 |
| 超时处理 | 弹窗提示"今日青少年模式使用时间已到"，无法继续使用 |
| 夜间禁用 | 22:00 — 06:00 无法使用（进入 App 直接显示"夜间休息中"页面） |
| 时间计算 | 跨越 22:00 时自动锁定；每日 06:00 重置 |

### 3.2 功能限制

| 功能 | 普通模式 | 青少年模式 |
| --- | --- | --- |
| 浏览 feed | ✅ | ✅（受控内容） |
| AI 文本造梗 | ✅ | ❌ 禁止 |
| AI 梗图生成 | ✅ | ❌ 禁止 |
| AI 视频生成 | ✅ | ❌ 禁止 |
| 发布梗卡 | ✅ | ❌ 禁止 |
| 评分 | ✅ | ❌ 禁止 |
| 评论 | ✅ | ❌ 禁止 |
| 私聊 | ✅ | ❌ 禁止 |
| 军团 | ✅ | ❌ 禁止 |
| PK | ✅ | ❌ 禁止 |
| 转发/分享 | ✅ | ❌ 禁止 |
| 付费（Pro/视频包） | ✅ | ❌ 禁止 |
| 搜索 | ✅ | ⚠️ 受限（仅预设分类）|
| 设置 | ✅ | ✅（仅关闭青少年模式、注销账号）|

### 3.3 退出青少年模式

- 需输入 4 位数字密码（首次开启时设置）
- 连续输入错误 5 次后锁定 10 分钟
- 退出后恢复全部功能

## 4. 数据库字段就位

### 4.1 users 表相关字段（已在 schema.sql 中就位）

user_profiles 表已有字段：
- `age_verified` BOOLEAN NOT NULL DEFAULT false
- `birth_date` DATE
- `is_teen_mode` BOOLEAN NOT NULL DEFAULT false
- `teen_pin_hash` TEXT

### 4.2 技术说明

- `birth_date` 仅在注册时填写
- 系统根据 `birth_date` + 当前日期动态计算年龄，不存储年龄字段
- `is_teen_mode` 默认值取决于年龄：< 18 岁默认 true
- `teen_pin_hash` 用 bcrypt 哈希

## 5. API 接口设计

### 5.1 GET /users/me/teen-status

返回当前用户的青少年模式状态（剩余时长、是否夜间禁用等）。

### 5.2 POST /users/me/teen-mode

开启/关闭青少年模式（请求含密码验证）。

## 6. RN 客户端实现要点

### 6.1 App 启动时检查

```
App 启动 → 检查 is_teen_mode
  → 如果 is_teen_mode = true:
    → 检查 is_night_disabled（22:00-06:00）
    → 如果夜间 → 显示"夜间休息中"页面
    → 否则 → 检查 remaining_minutes
    → 如果 ≤ 0 → 显示"今日时间已到"
    → 否则 → 正常进入（功能受限）
  → 如果 is_teen_mode = false: 正常进入
```

### 6.2 功能入口屏蔽

青少年模式下需屏蔽的入口：
- 底部 Tab "造梗" Tab 隐藏
- Feed 卡片上的评分按钮隐藏
- 梗卡详情页的评论输入框隐藏
- 我的页中 Pro 会员入口隐藏
