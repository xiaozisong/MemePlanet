## Instructions

- Scaffold a new Vite project with the official CLI.
- Install dependencies and run the dev server.
- Verify HMR by editing a module.

### Example

```sh
npm create vite@latest my-app
cd my-app
npm install
npm run dev
```

### Example

```sh
# pnpm
echo "Using pnpm"
pnpm create vite my-app
pnpm install
pnpm dev
```

### Notes

- Use Node LTS.
- Select the correct framework template.

Reference: https://cn.vitejs.dev/guide/
